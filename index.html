VSCode.NET

VSCode.NET is a language server for Visual Studio Code written in C#. It is designed to be compatible with the Microsoft/vscode-languageserver-node client. Currently, VSCode.NET supports .NET 4.5, 4.5.1, 4.5.2, and .NET Core 1.0.
